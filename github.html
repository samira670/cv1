  • K-Means Clustering

    This code demonstrates the application of the K-means clustering algorithm, beginning with data preparation and proceeding to the execution of K-means to identify clusters in the data.   • Hierarchical clustering

    The code provides a detailed demonstration of hierarchical clustering, a method used to group data points into clusters based on their similarity. The process involves preparing the data, generating a dendrogram to visualize the clustering process, and implementing the AgglomerativeClustering method from scikit-learn. Different linkage methods like 'ward', 'average', and their effectiveness with various distance metrics are evaluated using accuracy scores. This approach helps in understanding the impact of different clustering strategies on the grouping efficiency.

  • Explanatory Factor Analysis

    The code demonstrates Exploratory Factor Analysis (EFA) using scikit-learn to reduce dimensionality and identify underlying factors in a dataset. It details the process of setting up the data for analysis, applying the FactorAnalysis method, and displaying the factor loadings, which illustrate the contribution of each original variable to the identified factors. This approach helps in simplifying complex data by extracting essential features that explain patterns in the data.

  • Principal Component Analysis (PCA)

    The code illustrates how to perform Principal Component Analysis (PCA) using Python libraries. It begins by importing necessary packages such as pandas, numpy, and sklearn. The notebook loads a dataset, applies PCA using sklearn's PCA function, and fits the model to reduce the dataset's dimensionality. The principal components are extracted to explain the variance in the data, and results are visualized through plots and heatmaps to display the contribution of each component. This PCA application helps in understanding data structures and reducing features without losing significant information, ideal for data pre-processing or exploratory data analysis.

  • Association Rule Mining Using Apriori Algorithm

    The code demonstrates the application of the Apriori algorithm for association rule mining. The notebook then uses these itemsets to generate association rules based on metrics like confidence and lift, filtering rules that meet certain thresholds.   • K- Nearest Neighbors(KNN)

    This code implements a k-Nearest Neighbors (k-NN) classification using the mtcars dataset. It loads the data, selects four features (mpg, disp, hp, wt) as predictors, and uses am (automatic/manual transmission) as the target variable. The data is standardized and split into training and test sets. A KNeighborsClassifier is trained on the data, and predictions are made on the test set. The model's performance is evaluated using precision, recall, and F1-score, with an accuracy of 86%, demonstrating a simple instance-based learning approach.

  • Nive Bayes Classifier

    This code implements three types of Naive Bayes classifiers (Bernoulli, Multinomial, and Gaussian) to detect spam using the UCI Spambase dataset. The data is loaded, split into features and labels, and further divided into training and test sets. Each classifier is trained on the training data, and predictions are made on the test set. The models' performance is evaluated using accuracy, with the Bernoulli Naive Bayes classifier (with binarize=0.1) achieving the highest accuracy of 91%, demonstrating the effectiveness of Naive Bayes for spam detection tasks.

  • Enseble Learning with Random forest

  • Stroke Prediction

    In the stroke prediction project hosted on GitHub, a variety of machine learning models were employed to predict the likelihood of strokes based on health-related data from individuals. The dataset used includes numerous parameters such as age, gender, hypertension status, and smoking habits, which are integral to assessing stroke risk. Among the models tested, which include Logistic Regression, Random Forest, XGBoost, and Gradient Boosting, the Random Forest classifier emerged as the most effective. It achieved an outstanding accuracy of 99%, along with precision and recall, making it the best model for this particular dataset. This superior performance can be attributed to Random Forest's ability to handle imbalanced datasets and its robustness against overfitting, proving it to be an invaluable tool in the predictive analysis of medical conditions.
