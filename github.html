  • Aero engine defect Classification (ViT)

    Dataset: Dataset. This project develops a machine learning model to classify aero-engine defects by utilizing a pre-trained DeiT model for feature extraction, UMAP for dimensionality reduction, and an ensemble of Random Forest and XGBoost classifiers. The process includes downloading and preparing the dataset, extracting image features with the DeiT model, and visualizing these features using UMAP. The data is split into training and validation sets, with both classifiers being trained and combined into an ensemble model. The ensemble model is then evaluated using precision, recall, and F1-score metrics, and its predictions are interpreted with SHAP to visualize feature importance. Additionally, various functions are defined to visualize the top activations and images with bounding boxes based on SHAP values, providing a comprehensive understanding of the model's decision-making process.
